Go (Golang) ​

  • 版本：Go 1.26.0
  • 框架：[Echo](https://echo LAB.com/) v5.0.4 — 高效能 HTTP 框架

  • 應用場景：
    • 營建工程管理 ERP 系統後端 API
    • RESTful API 設計與開發
    • 資料庫操作與管理
    • 系統架構設計與重構
