## 伺服器特色

| 類型 | 內容 |
|------|------|
| 遊戲模式 | 插件生存、24 小時開服 |
| 主要版本 | 1.12.2，也曾支援 1.19.4 等版本 |
| 核心系統 | Mcmmo 生活技能、住宅區、公會、寵物 |
| 社群機制 | 優質玩家驗證、新手引導、規則與指令教學 |
| 經營方式 | 自架服務、網站維護、Discord 與 Facebook 社群支援 |

::: info 經營重點
這個伺服器不只是遊戲專案，也是一個長期練習技術、營運、文件整理與社群溝通的實戰場域。
:::

## 站務經驗

- **伺服器架設**：處理版本、插件、資料備份、網路與硬體資源。
- **玩家支援**：回應新手問題、管理規則、協助帳號與遊戲內權限。
- **文件維護**：撰寫規則、指令教學、驗證流程與伺服器介紹。
- **社群經營**：維護 Discord、Facebook 粉絲專頁與社團。
- **內容產出**：製作宣傳圖片、影片與 YouTube 內容。
