## 後端開發

### Go (Golang)

主要使用的後端程式語言，在工作中大量使用 Golang 進行 ERP 系統的 Web 化重構。

- **版本**：Go 1.26.0
- **框架**：[Echo](https://echo LAB.com/) v5.0.4 — 高效能 HTTP 框架
- **ORM**：[XORM](https://xorm.io/) v1.3.11 — 物件關聯對應工具
- **應用場景**：
  - 營建工程管理 ERP 系統後端 API
  - RESTful API 設計與開發
  - 資料庫操作與管理
  - 系統架構設計與重構

## 前端開發

### Vue.js 生態系統

現代化前端開發以 Vue.js 為核心，搭配多個工具與框架。

- **框架**：Vue.js v3.5.29（Composition API）
- **建構工具**：Vite v7.2.6
- **UI 框架**：
  - [PrimeVue](https://primevue.org/) v4.5.4 — 豐富的 UI 元件庫
  - [TailwindCSS](https://tailwindcss.com/) v4.1.18 — Utility-first CSS 框架
  - [MasterCSS](https://master.co/) v1.37.8 — 語法驅動的 CSS 框架
- **HTTP 客戶端**：Axios v1.13.5
