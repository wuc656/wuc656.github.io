後端 ​

Go目前主力後端語言，用於 ERP Web 化、RESTful API、資料處理與系統整合。
EchoGo Web 框架，適合快速建立 API、中介層與後端服務。
XORMORM 工具，用於資料表映射、查詢封裝與維護資料存取層。
Delphi 2009維護既有桌面系統，也作為新舊系統行為對齊的重要參考。

前端 ​

Vue 3目前主力前端框架，偏好 Composition API，適合後台系統與模組化 UI。
Vite前端建置工具，負責快速開發、熱更新與專案打包。
PrimeVue後台系統常用 UI 元件庫，適合資料表、表單、彈窗與操作型介面。
TailwindCSS / MasterCSS用於快速建立一致的版面、間距與響應式樣式。

工作流 ​

  1. 先理解既有系統資料流與使用者操作流程。
  2. 將可拆分的功能轉成 API 與前端頁面。
  3. 保留可對照的舊系統行為，降低重構過程的風險。
  4. 逐步整理共用元件、資料存取層與文件。

注意

我目前的重點不是一次性重寫所有系統，而是讓既有 Delphi 系統能穩定維護，同時把適合 Web 化的功能逐步搬到 Go + Vue 架構。
